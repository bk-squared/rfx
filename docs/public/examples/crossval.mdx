Cross-validation scripts test solver results; they are not introductory
examples. Follow the [examples learning path](/rfx/examples/) first, then use
the [Cross-Solver Validation](/rfx/validation/cross-solver/) page to choose a
check.

## Repository checkout

In a source checkout, `validation/README.md` lists every validation script, its
reference calculation, and the meaning of its exit code.

A passing script supports only its stated geometry, mesh, frequency range,
source, and output. It does not validate a different RF configuration.

## Two setup rules that matter

1. Model the ground plane as geometry.
2. Use a physically appropriate absorber setup for the reference solver.

Those are the two easiest ways to turn a patch-antenna comparison into a false
comparison.

## What not to promote

- older narrative examples without crossval evidence
- convenience snippets that skip the finite-ground-plane detail
- shadow-lane results without a support-boundary note
- superseded example scripts when a newer committed crossval exists for the same
  workflow family
- crossval scripts that are intentionally not part of the current promoted
  public example set

This page is the practical example hub for public RF workflows.

## Current recommended workflow

| Stage | Goal | Example |
|---|---|---|
| 1. Build the geometry | describe the RF structure explicitly | `validation/crossval/05_patch_antenna.py` |
| 2. Extract resonance | use a clean probe / Harminv path | patch workflow notes |
| 3. Cross-check when needed | compare against the current repo reference path | crossval script output |
| 4. Interpret the result | separate design estimate from solver result | validation and limitation pages |

## Current public examples

- `validation/crossval/05_patch_antenna.py` — practical patch antenna cross-check
- `validation/crossval/11_waveguide_port_wr90.py` — rectangular waveguide-port workflow
- `examples/tutorials/nonuniform_patch_demo.py` — non-uniform mesh demo; limited-support path
- `examples/inverse_design/multilayer_ar_coating.py` — conservative inverse-design demo

## Public example criteria

- use paths that exist in the current repository
- include the modeling assumptions
- state the support boundary for the result
- keep local run artifacts out of the public docs

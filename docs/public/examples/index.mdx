The public example surface is **crossval-first**.
Prefer the newest validated cross-solver evidence over older narrative examples
unless a legacy example is explicitly promoted.

## Recommended entry points

| Page or script | Why it matters |
|---|---|
| [Crossval First](./crossval/) | the newest validated finite-ground-plane patch evidence |
| [`rf-workflows`](./rf-workflows/) | the current practical workflow hub for validated RF examples |
| `examples/crossval/12_patch_antenna.py` | newest public patch reference using validated cross-solver evidence |
| `examples/40_accuracy_validation/run_all.py` | published benchmark ladder for the core RF cases |

## How to read the example surface

- Use crossval scripts when you need a reproducible public reference.
- Use benchmark scripts when you need the strongest numerical claim.
- Treat older narrative examples as legacy unless they are tied to newer
  validation evidence.

## Current public default

If you need a single example to cite first, use the crossval patch reference
before older tutorial narratives.

## Public rule

If an example is not validated, say so explicitly.
If an example is validated on a shadow lane, say that too.

## Overview

`add_floquet_port(...)` configures Bloch-periodic unit-cell excitation for periodic or phased-array style studies.

Current public framing:

- useful for exploratory periodic/unit-cell workflows
- not the default S-parameter path
- not a replacement for the recommended uniform Yee RF examples

See [Sources & Ports](../sources-ports/) for the general port framework and [Waveguide Ports](../waveguide-ports/) for rectangular waveguide modes.

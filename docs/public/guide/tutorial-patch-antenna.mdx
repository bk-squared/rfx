## Overview

This tutorial will walk through a complete patch antenna design workflow:

1. **Specification** — target frequency, substrate, and form factor
2. **Geometry setup** — ground plane, substrate, patch, and feed
3. **Simulation** — auto-configured FDTD run with probe placement
4. **Analysis** — resonance extraction, impedance matching, radiation pattern

For a quicker introduction, see [Your First Patch Antenna](../first-patch/).

## Overview

This tutorial will walk through a microstrip bandpass filter design:

1. **Specification** — center frequency, bandwidth, and filter order
2. **Coupled-line geometry** — line widths, gaps, and lengths from synthesis
3. **Simulation** — two-port S-parameter extraction
4. **Analysis** — insertion loss, return loss, and passband ripple

See [Sources & Ports](../sources-ports/) for port setup and [Probes & S-Parameters](../probes-sparams/) for S-matrix extraction.

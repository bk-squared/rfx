## Overview

This tutorial will demonstrate how to verify simulation accuracy via convergence studies:

1. **Baseline simulation** — coarse mesh at the CFL limit
2. **Mesh refinement** — progressive halving of cell size
3. **Metric tracking** — resonant frequency, S-parameter, and field convergence
4. **Convergence criterion** — when results stabilise within a tolerance

See [Non-Uniform Mesh](../nonuniform-mesh/) for graded profiles and [Validation](../validation/) for the cross-simulator benchmark suite.

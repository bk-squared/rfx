## Overview

This placeholder tracks rfx's experimental subgridding research lane:

- **SBP-SAT coupling** — summation-by-parts operators with simultaneous approximation terms under active validation
- **Local refinement** — higher resolution in regions of interest without refining the entire domain
- **JIT compilation** — subgridding operators are compiled via JAX for GPU performance
- **Use cases** — fine features (vias, slots, feed gaps) embedded in electrically large structures

See [Non-Uniform Mesh](../nonuniform-mesh/) for the simpler graded-profile alternative.

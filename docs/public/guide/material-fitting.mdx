## Overview

This guide will cover rfx's material fitting capabilities:

- **CSV import** — load measured ε(f) / μ(f) data from standard formats
- **Debye fitting** — automatic pole extraction for wideband dielectric models
- **Lorentz fitting** — resonant material models for metamaterials and ferrites
- **FDTD-based fitting** — differentiable simulation loop for gradient-based parameter refinement

See [Materials & Geometry](../materials-geometry/) for the built-in material library and dispersive model usage.

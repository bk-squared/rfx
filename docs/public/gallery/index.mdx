A visual tour of structures, fields, outputs, and differentiable calculations.
The slab and waveguide pages state their reference comparisons and supported
gradient scope. The patch page is explicitly diagnostic because its stored port
setup does not pass current preflight; its plots and derivative are not RF
validation. Read the status and limits on each page before reusing a number.

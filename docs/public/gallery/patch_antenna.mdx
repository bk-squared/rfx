## What this is

A half-wavelength rectangular microstrip patch on an FR4 substrate over a finite
ground plane, fed by a single coaxial-style probe. It is the workhorse printed
radiator for the 2.4 GHz band. The resonant length is set by the half-wave
condition in the substrate's effective permittivity,

$$
L \;\approx\; \frac{c}{2 f_0 \sqrt{\varepsilon_\text{eff}}} \;-\; 2\,\Delta L .
$$

## What you'll learn

- Building a **finite ground plane** geometry (PEC box below the substrate, not an infinite `pec_face`) so the structure radiates instead of behaving as a closed cavity.
- Closing the open domain with a **6-face CPML** absorber.
- Extracting the resonant frequency from the ring-down with **Harminv** filter diagonalisation — independent of port calibration.
- Driving a **50 Ω lumped port** and reading the **|S11|** return-loss response.
- Cross-validating the FDTD resonance against the **first-order transmission-line** estimate.

## Geometry & materials

The design follows the standard transmission-line synthesis (Bahl & Trivedi).
Express the dimensions as code so they stay tied to the design frequency:

```python
import numpy as np
C0 = 299_792_458.0

f0    = 2.4e9      # design frequency
eps_r = 4.4        # FR4 relative permittivity
h     = 1.6e-3     # substrate thickness
tan_d = 0.02       # FR4 loss tangent at 2.4 GHz

W       = C0 / (2 * f0) * np.sqrt(2 / (eps_r + 1))           # patch width
eps_eff = (eps_r + 1) / 2 + (eps_r - 1) / 2 * (1 + 12 * h / W) ** -0.5
dL      = 0.412 * h * ((eps_eff + 0.3) * (W / h + 0.264)) \
                    / ((eps_eff - 0.258) * (W / h + 0.8))    # fringing extension
L       = C0 / (2 * f0 * np.sqrt(eps_eff)) - 2 * dL          # patch length

# FR4 conductivity from the loss tangent
sigma_fr4 = 2 * np.pi * f0 * 8.854e-12 * eps_r * tan_d
# -> W = 38.0 mm, L = 29.4 mm, eps_eff = 4.09, dL = 0.74 mm
```

The patch length comes out to L = 29.4 mm and width W = 38.0 mm on a
1.6 mm FR4 board with ε_eff ≈ 4.09. The probe feed is inset from the radiating
edge to land near a 50 Ω input.

## Source, ports, boundaries, mesh

- **50 Ω lumped port** at the inset feed — it loads the structure with a
  reference impedance so the reflection is read directly as |S11|.
- **6-face CPML, ≥ 20 cells.** The patch radiates into all six faces, so every
  face is absorbing. A thin CPML reflects low-frequency near-field energy back
  in, which closes the domain into a cavity and pushes the resonance high; a
  thick absorber keeps the boundary transparent down to band edge.
- **λ/20 mesh.** Twenty cells per wavelength in the substrate keeps the
  half-wave field variation along L well sampled and the dispersion error small.
- **Finite ground plane.** The ground is an explicit finite PEC box beneath the
  substrate; the half-space below it radiates into the bottom CPML, which is the
  correct picture for a real patch.

<Aside type="note">
An infinite ground plane (a `pec_face` at the domain floor) turns the structure
into a closed cavity and biases the resonance several percent high. Use a finite
PEC box.
</Aside>

## Run

The construction below mirrors `examples/crossval/05_patch_antenna.py`. A
broadband Ez source drives the patch; the probe records the ring-down for
Harminv. A non-uniform z-profile resolves the thin substrate without inflating
the cell count.

```python
import numpy as np
from rfx import Simulation, Box
from rfx.sources.sources import GaussianPulse
from rfx.auto_config import smooth_grading

# --- domain: finite ground plane + air margins, non-uniform z ---
dx, n_sub = 1.0e-3, 6
dz_sub    = h / n_sub
air_below, air_above = 12.0e-3, 25.0e-3
gx, gy = 60.0e-3, 55.0e-3                     # finite ground plane
dom_x, dom_y = gx + 20e-3, gy + 20e-3
dom_z = air_below + h + air_above

raw_dz = np.concatenate([
    np.full(int(np.ceil(air_below / dx)), dx),   # air below the ground plane
    np.full(1, dz_sub),                           # ground-plane PEC cell
    np.full(n_sub, dz_sub),                       # substrate
    np.full(int(np.ceil(air_above / dx)), dx),    # air above the patch
])
dz_profile = smooth_grading(raw_dz, max_ratio=1.3)

sim = Simulation(
    freq_max=4e9,
    domain=(dom_x, dom_y, 0),
    dx=dx,
    dz_profile=dz_profile,
    boundary="cpml",
    cpml_layers=20,
)

# --- materials and geometry ---
sim.add_material("fr4", eps_r=eps_r, sigma=sigma_fr4)

z_gnd_lo = air_below - dz_sub
z_sub_lo, z_sub_hi = air_below, air_below + h
gx_lo, gy_lo = (dom_x - gx) / 2, (dom_y - gy) / 2
patch_x_lo, patch_y_lo = dom_x / 2 - L / 2, dom_y / 2 - W / 2

sim.add(Box((gx_lo, gy_lo, z_gnd_lo), (gx_lo + gx, gy_lo + gy, z_sub_lo)), material="pec")   # ground
sim.add(Box((gx_lo, gy_lo, z_sub_lo), (gx_lo + gx, gy_lo + gy, z_sub_hi)), material="fr4")   # substrate
sim.add(Box((patch_x_lo, patch_y_lo, z_sub_hi),
            (patch_x_lo + L, patch_y_lo + W, z_sub_hi + dz_sub)), material="pec")            # patch

# --- broadband drive + probe for Harminv ring-down ---
feed = (patch_x_lo + 8e-3, dom_y / 2, z_sub_lo + dz_sub * 2.5)
sim.add_source(feed, component="ez", waveform=GaussianPulse(f0=f0, bandwidth=1.2))
sim.add_probe((dom_x / 2 + 5e-3, dom_y / 2 + 5e-3, feed[2]), component="ez")

result = sim.run(num_periods=60)
modes  = result.find_resonances(freq_range=(1.5e9, 3.5e9))
best   = min(modes, key=lambda m: abs(m.freq - f0))
print(f"resonance {best.freq/1e9:.3f} GHz, Q {best.Q:.0f}")
```

Run it from the command line:

```bash
python examples/crossval/05_patch_antenna.py
```

## Validation & limits

The FDTD Harminv resonance is **2.66 GHz**; the first-order transmission-line
estimate is **2.42 GHz** — about +10 % apart.

| Quantity | Frequency | Note |
|---|---|---|
| rfx Harminv (ring-down) | **2.66 GHz** | full-wave; Q ≈ 30 |
| Analytic (transmission-line) | **2.42 GHz** | first-order, no finite-ground term |
| Difference | +9.8 % | analytic underestimates |

![rfx Harminv resonance vs first-order analytic estimate](/rfx/gallery/assets/patch_antenna/validation.png)

The first-order formula biases **low**: its closed-form ΔL undercounts the
fringing extension, and it ignores the finite-ground and probe-feed inductance
that lengthen the electrical path. The full-wave solver carries the true
boundary conditions, so for the as-built electrical length it is the more
trustworthy of the two; the analytic value is a sanity bound, not the reference.

**Read the resonance from the ring-down, not from the |S11| dip.** The dip and
the resonance are different frequencies. The single-cell lumped port carries a
parasitic cell reactance that adds a monotonic background to |S11|, so the
impedance-match minimum (here near 3.1 GHz, shallow at about −0.8 dB) is a
secondary diagnostic — the Harminv ring-down at 2.66 GHz is the resonance.

**Convergence.** Tightening the absorber from 12 to 20 CPML cells and refining
the mesh from λ/20 to λ/30 shifts the extracted resonance by under 1 %, so the
2.66 GHz figure is converged with respect to both the boundary and the grid.

X-band WR-90 rectangular waveguide port S-matrix (TE10, 8.2–12.4 GHz) for a
matched, empty guide, computed with the canonical `add_waveguide_port` +
`compute_waveguide_s_matrix` pipeline.

<Aside type="tip" title="Validation — tier E5">
**Metric:** `max|S11| < 0.02`, `min|S21| > 0.97` vs the analytic matched-load
reference (`|S11| = 0`, `|S21| = 1`).
**Tolerance:** `max|S11| < 0.02`, `min|S21| > 0.97`.
**Reference solver:** analytic matched-load (MEEP / OpenEMS / Palace
cross-check on VESSL).
</Aside>
